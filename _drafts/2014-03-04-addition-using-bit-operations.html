Lets look at the algorithm first ::
Assume we got numbers m and n to add.
1. m Xor n = X
2. Now get carry by doing AND:  m AND n = C.  Shift C to left. C = C << 1. If C is zero then stop - result is in X else go to step 3.
3. m =X, n= C. Go to 2.
//Here is the code
int add(int m, int n)
{
    int result = 0;
    int carry = 0;
    do
    {
        result = m^n;
        carry = m&n;
        carry = carry << 1;
        m = result;
        n = carry;
    }while(carry!=0);
    return result;
}
Example -
m = 100011, n = 011011
m ^ n = 111000 = X
m AND n = 000011 = C,  C = 000011 << 1 = 000110
X ^ C = 111000 ^ 000110 = 111110
X AND C = 0 - Stop.
